# Nginx Reverse Proxy

A beginner-friendly Linux System Administration project to learn how to install and configure Nginx as a web server and reverse proxy on Ubuntu Server.

## Project Objectives

- Install Nginx
- Configure Nginx Web Server
- Configure Reverse Proxy
- Manage Nginx Service
- Host a Simple Website

## Technologies Used

- Ubuntu Server 24.04 LTS
- Nginx
- HTML
- Git & GitHub

## Commands Used

- `sudo apt update`
- `sudo apt install nginx`
- `systemctl status nginx`
- `systemctl start nginx`
- `systemctl restart nginx`
- `systemctl enable nginx`
- `nginx -t`
